# 📱 Redmi – Complete Information

## 🧭 Overview
**Redmi** is a sub-brand of **Xiaomi** (Chinese electronics company).  
It was launched in **July 2013** as a budget smartphone line, but later became an independent brand for mid-range and affordable devices.  
In 2019, Redmi was spun off as a separate brand under Xiaomi, focusing on **"flagship killer"** specifications at lower prices.

---

## 📅 History & Milestones
- **2013** – First Redmi phone (Redmi 1) launched – budget 4.7" device.
- **2014** – Redmi Note series started (large screen, big battery).
- **2016** – Redmi Pro – first with dual cameras.
- **2017** – Redmi 5/5 Plus – 18:9 aspect ratio trend.
- **2019** – Redmi Note 7 – 48MP camera, iconic design.
- **2020** – Redmi Note 9 series – huge success globally.
- **2021** – Redmi K series (K40) – flagship specs under ₹30,000.
- **2023** – Redmi Note 12 series – 200MP camera, 120W charging.
- **2024** – Redmi 13 series & HyperOS 2.0 rollout.

## 🛠️ Tips for Redmi Users

### 🔄 Enable Developer Options
- Go to **Settings → About Phone → Tap MIUI/HyperOS Version 7 times**
- Then **Additional Settings → Developer Options** – to enable USB debugging, animation scale, etc.

### 📦 Remove Bloatware
- You can uninstall or disable many pre-installed apps via Settings or ADB
- Use `adb shell pm uninstall -k --user 0 <package-name>` to remove unwanted system apps

### 🚀 Boost Performance
- Disable animations (Developer Options → Window/Transition/Animator scale → 0.5x or Off)
- Use **Game Turbo** mode for gaming
- Clear cache regularly

### 🔋 Battery Optimization
- Use **Dark Mode** (AMOLED screens save power)
- Turn off 5G if not needed (switch to 4G)
- Use battery saver mode in extreme cases

### 📸 Camera Tips
- Use **Pro mode** for manual control (ISO, shutter speed, focus)
- Enable **Google Camera (GCam)** for better image processing
- Use night mode in low light

## 🔧 Common Issues & Fixes

### ❌ Phone heating up
- Close background apps
- Don't use during charging
- Update to latest OS version
- Use lighter case (avoid thick covers)

### ❌ Battery draining fast
- Turn off Always‑On Display (if available)
- Disable auto-sync for unnecessary apps
- Check battery usage in settings

### ❌ Camera not focusing
- Clean lens with microfiber cloth
- Restart camera app
- Reset camera settings

### ❌ Phone restarting randomly
- Clear cache partition (Recovery mode)
- Uninstall recently installed suspicious apps
- Factory reset if persistent
